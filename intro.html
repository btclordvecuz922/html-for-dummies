HTML Full Course for Beginners

This page contains a complete beginner-friendly HTML course. If you study everything here, you will understand more than 60% of HTML.

1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

HTML is not a programming language. It is a markup language used to structure content.

2. How Websites Work

  1. You type a website address in your browser
  2. The browser requests the page from a server
  3. The server sends HTML files
  4. The browser reads the HTML and displays the page

3. Basic Structure of an HTML Document

Every HTML page follows this structure:


<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first website</p>
</body>
</html>

5. Headings

HTML has six heading levels:


<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Small Heading</h3>
<h4>Smaller Heading</h4>
<h5>Very Small Heading</h5>
<h6>Smallest Heading</h6>

6. Paragraphs and Text Formatting


<p>This is a paragraph</p>
<b>Bold text</b>
<i>Italic text</i>
<strong>Important text</strong>

7. Links

Links allow users to move between pages.


<a href="https://google.com">Visit Google</a>
<a href="about.html">About Page</a>

8. Images


<img src="image.jpg" alt="My Image">

9. Lists

Unordered List (Dots)


<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered List (Numbers)


<ol>
  <li>Install editor</li>
  <li>Create HTML file</li>
  <li>Write code</li>
</ol>

10. Attributes

Attributes give extra information to elements.


<p style="color:red;">Red text</p>
<a href="#" title="Click me">Link</a>

11. HTML Comments


<!-- This is a comment -->

12. Best Practices

  • Always close your tags
  • Indent your code properly
  • Use meaningful file names
  • Practice regularly
