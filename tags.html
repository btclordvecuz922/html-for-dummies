HTML Tags for Beginners

This page explains all the basic HTML tags you need to know as a beginner.

1. Basic Structure Tags

  • <!DOCTYPE html> - Declares HTML version
  • <html> - Root of the HTML document
  • <head> - Contains meta info, title, links
  • <title> - Page title displayed in browser tab
  • <body> - All visible content goes here

2. Headings & Text

  • <h1> - <h6> - Headings
  • <p> - Paragraph
  • <b> - Bold text
  • <i> - Italic text
  • <u> - Underlined text
  • <strong> - Important text
  • <em> - Emphasized text
  • <br> - Line break
  • <hr> - Horizontal line

3. Links & Media

  • <a href="url"> - Link
  • <img src="image.jpg" alt="text"> - Image
  • <video> - Video
  • <audio> - Audio

4. Lists

  • <ul> - Unordered list (dots)
  • <ol> - Ordered list (numbers)
  • <li> - List item

5. Forms & Input

  • <form> - Form wrapper
  • <input> - Input fields
  • <textarea> - Multi-line text
  • <button> - Button
  • <select> - Dropdown
  • <label> - Label for input

6. Semantic Tags

  • <header> - Page header
  • <footer> - Page footer
  • <section> - Section wrapper
  • <article> - Article content
  • <nav> - Navigation
  • <main> - Main content
  • <aside> - Sidebar info

7. Comments

  • <!-- comment --> - Comment not displayed in browser

Now you know all the basic HTML tags! Next, visit the Examples page to see them in action.
